# 🎮 Tic Tac Toe - Termux Elite Edition

An **elite, professional, and mobile-friendly Tic Tac Toe game** in Python. Designed for fun, clarity, and replayability, with both a **modular multi-file version** and a **standalone single-file version**.

---

## 🚀 Features

- **Modes:** Human vs Human & Human vs AI  
- **AI:** Smart AI that blocks and wins intelligently  
- **Board:** Single board redraw with numbered placeholders (1-9)  
- **Winning Highlights:** Color-highlighted winning combination  
- **Score Tracking:** Keep scores across multiple rounds  
- **Replay Option:** Play again without overlapping previous moves  
- **Animations & Effects:** Smooth in-terminal updates with `rich`  
- **Mobile-Friendly:** Optimized for Termux on Android  


---


## 💻 Installation

1. **Clone the repository:**
   
   ```bash
   git clone https://github.com/YourUsername/tic-tac-toe.git
   cd tic-tac-toe
   ```

3. **Install dependencies:**
   
   ```bash
   pip install -r requirements.txt
   ```
   
3. **Navigate to your desired version:**
   
   ```bash
   cd modular    # for multi-file version # or
   cd standalone # for single-file version
   ```
   
4. **Run the Game:**
   
   ```bash
   python main.py       # modular
   python tictactoe.py  # standalone
   ```

---


## 🎮 How to Play

- Choose a mode: Human vs Human or Human vs AI
- Enter a number 1-9 to place your marker (X or O)
- Watch the board update in-place
- Winning combo is highlighted in color
- View scores after each round
- Choose to replay or exit


---


## ⚡ Requirements

- Python 3.x
- rich library for colored board and animations

**Install with:**
```bash
pip install rich>=14.0.0
```
